В [[Rust|Rust]], как и в других языках, существуют базовые управляющие конструкции.

## If / Else

**Пример использования if/else:**

```Rust
if x > 5 {
    println!("Больше 5");
} else {
    println!("Меньше или равно 5");
}
```
