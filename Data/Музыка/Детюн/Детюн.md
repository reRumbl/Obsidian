**Детюн** - это частота ноты `A4`, относительно которой строятся частоты всех остальных нот. По умолчанию детюн составляет `440Hz`. Детюн можно изменить, чтобы добиться экспериментального и непривычного человеческому уху звучания.

## Изменение детюна

Менять детюна нужно у каждого музыкального инструмента по отдельности. Изменение детюна происходит двумя способами:

- Напрямую (в настройках инструмента).

- Ручное изменение частоты при помощи питча.

### Прямое изменение детюна

В настройках инструмента требуется найти параметр **detune** и изменить его с `440Hz` на желаемый.

*Пример: в Serum параметр **detune** находится на вкладке Global.*

### Ручное изменение детюна

В некоторых инструментах, а также семплах нельзя просто так поменять детюн. В таком случае приходится использовать любое изменение питча (через DAW или плагины).

*Пример: в FL Studio достаточно нажать на семпл и изменить питч при помощи ручки `channel pitch`.*

**Таблица популярных детюнов и соответствующих им питчей:**

| Детюн  | Питч     |
| ------ | -------- |
| 428 Hz | -50 cent |
| 430 Hz | -40 cent |
| 432 Hz | -32 cent |
| 448 Hz | +31 cent |
| 450 Hz | +39 cent |
| 453 Hz | +50 cent |
