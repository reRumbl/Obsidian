**Определение жанра музыки** - художественная задача, которая стоит перед продюсером. Правильные жанры помогают продвигать музыку.

## BPM

| BPM     | Жанры                    |
| ------- | ------------------------ |
| 80-120  | hip-hop / boom bap       |
| 130-150 | trap / rage / hiphop     |
| 160-180 | jungle / breakcore / dnb |

## Паттерн Драмки

| Паттерн  | Жанры              |
| -------- | ------------------ |
| Ровный   | trap               |
| Ломанный | break / breakcore  |
| Качающий | hip hop / boom bap |

## Атмосфера / Вайб

Какая общая атмосфера в треке?

| Вайб        | Жанры              |
| ----------- | ------------------ |
| грустный    | sad / emo          |
| агрессивный | rage / dark        |
| воздушный   | ambient / cloud    |
| грязный     | grunge / distorted |
