Классическая модель Lockheed Martin:

1. Reconnaisance (Разведка): Сбор данных о жертве (сканирование Nmap).

2. Weaponization (Вооружение): Создание вредоносного ПО (подготовка эксплойта).

3. Delievery (Доставка): Отправка вредоносного ПО жертве (почта, флешка, загрузка файла).

4. Expoitation (Эксплуатация): Срабатывание вредоносного ПО (использование уязвимостей).

5. Installation (Установка): Закрепление в системе (создание пользователя-администратора, установка бэкдора).

6. Command & Control (C2 / Управление): Связь зараженной машины с сервером хакера (Reverse Shell).

7. Actions on Objectives (Действия над целью): Кража данных, шифрование (Ransomware)
